Technical Philosophy

State Machines + LLMs

LLMs are powerful but unpredictable—they shouldn't orchestrate. I build explicit state machines as the control plane, with LLMs handling NLU/NLG within bounded contexts.

Safety as Infrastructure

Safety isn't a filter—it's architectural. Input sanitization, prompt injection defense, PII detection, and human handoff built into every layer.

Evaluation-Driven Development

Automated evaluation infrastructure (agent gyms, LLM-as-Judge, voice benchmarks) closes the loop between deployment and optimization.
