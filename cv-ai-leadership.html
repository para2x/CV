Technical Philosophy

State Machines + LLMs

LLMs are powerful but unpredictable—they shouldn't be the orchestrator. I build explicit state machines as the control plane: deterministic routing, transitions, and safety enforcement, with LLMs handling NLU/NLG within bounded contexts.

Safety as Infrastructure

Safety isn't a filter on outputs—it's built into every layer. Input sanitization, prompt injection defense, PII detection, escalation thresholds, and human handoff are architectural decisions, not afterthoughts.

Evaluation-Driven Development

If you can't measure it, you can't improve it. I build automated evaluation infrastructure (agent gyms, LLM-as-Judge, voice quality benchmarks) that closes the loop between deployment and optimization.
